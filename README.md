## How to Import the Workflow into n8n

1. **Open n8n editor**  
   Navigate to the n8n editor URL (e.g., `https://ai.educom.nu/t/<HTTP_PORT>/`).

2. **Import the workflow**  
   - Click the **Import** button (top‑right corner of the editor).  
   - Choose **Import from File**.  
   - Select the file `n8n/route1_workflow.json` from this repository.

3. **Adjust the CSV URL (if needed)**  
   The workflow uses the raw GitHub URL:  
   ```
   https://raw.githubusercontent.com/Loekie1103/educom-helpdesk-n8n-loek/main/tickets.csv
   ```  
   If you move the CSV to another location, edit the **HTTP Request** node → **URL** field accordingly.

4. **Run the workflow**  
   - Click **Execute Workflow**.  
   - The workflow will download `tickets.csv`, parse it, and store a preview of the first ticket in the `firstTicket` field (visible in the execution log).

5. **Verify the data**  
   After execution, open the **Execution List** → select the latest run → inspect the **Set - First ticket preview** node output. You should see the subject of the first ticket.
